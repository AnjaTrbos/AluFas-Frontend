# Alufas Frontend

React + Vite frontend for the Alufas workflow screens.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Styling

Tailwind CSS v4 is configured through [src/index.css](./src/index.css) using the CSS-first setup.
Semantic UI tokens such as `bg-primary`, `text-muted-foreground`, and the dialog/dropdown animation utilities are defined there.
